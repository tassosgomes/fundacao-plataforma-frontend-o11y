# fundacao-plataforma-frontend-o11y

## Exemplo executável

O repositório inclui uma demo completa de consumo da biblioteca em:

- `examples/`

Essa demo usa React + Vite com um Host e dois MFEs para demonstrar:

- inicialização centralizada no Host
- contexto de rota
- contexto de MFE
- spans manuais
- erro controlado
- visualização local no Jaeger

Guia de execução: `examples/README.md`
